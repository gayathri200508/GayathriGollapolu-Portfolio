---

## 🚀 Run Locally

```bash
# 1. Clone the repo
git clone https://github.com/gayathri200508/gg-portfolio.git
cd gg-portfolio

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run the app
streamlit run app.py
```

Open your browser at `http://localhost:8501` 🎉
